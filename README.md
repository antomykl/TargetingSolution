# Coordinate System Samples

Static demos for coordinate systems using SVG and Canvas.

## Structure

- `index.html` (root landing page)
- `CoordinateSystem/index.html`
- `CoordinateSystem/svg-sample.html`
- `CoordinateSystem/canvas-sample.html`

## Deploy with GitHub Pages

This repository includes `.github/workflows/deploy-pages.yml` to publish on each push to `main`.

Expected site URL after first successful deployment:

`https://<your-github-username>.github.io/<repository-name>/`
